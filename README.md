# 기술 스택
_____________

```r
-   React (Javascript library)
-   Redux (state management)
-   React Router (for routing)
-   Axios (for API calls)
-   Styled component
``` 


_1. 기능 소개_
___1. Login___
-로그인 기능 ..
___2.SignUp___
- 회원가입 기능.
___3. StoreList___
- 가게 불러오기(db)
- 가게 검색기능.
___4. OrderCheck___
- 해당가게의 메뉴 불러오기
- 주문표 기능.(해당메뉴 주문표등록, 수량, 합계계산..)
- 결제하기 
___5. OrderComplete___
- 선택한 메뉴 리스트 보여주기
- 총 결제금액 보여주기
- 해당 Id의 닉네임 보여주기.
